# 🐕 Walkydoo

Walkydoo is a web application designed for dog owners who love exploring new walking routes.

It helps users discover dog-friendly routes nearby or in other locations and share their favorite walks with the community.

# 🌍 Discover Routes

Find walking routes in your area or explore new paths while traveling.

# 🗺️ Share Your Favorite Walks

Registered users can draw and save their own routes directly on the map. Once saved, they become visible to all users.

# 🧰 Tech Stack

Walkydoo is built with the following technologies:

🐘 PHP – Full-stack application (frontend + backend logic)
🗺️ Leaflet – Interactive maps and route drawing
🌐 HTML / CSS – Structure and styling

# 💡 Features
Discover dog walking routes nearby or worldwide
Create and share your own routes
Interactive map drawing
Community-driven content
